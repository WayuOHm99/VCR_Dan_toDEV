# การวิเคราะห์ตารางแข่งขัน (Race Calendar Analysis)

ตารางการแข่งขันถูกจัดวางลำดับ (Periodization) ไว้อย่างเหมาะสมและสมเหตุสมผล โดยแบ่งเป็น A, B, และ C Race

## 1. หนองบุญมากมินิมาราธอน (5k)
- **วันที่:** 2 ส.ค. 2026
- **เป้าหมาย:** ทดสอบระบบเอาเวลา
- **ความน่าจะเป็น:** สูงมาก (Very High)
- **บทวิเคราะห์:** ระยะเวลา 4 สัปดาห์หลังจากการเทส VCR เป็นจังหวะที่เหมาะสมสำหรับการทำ Sharpening (ลับความเร็ว) เป็นสนามทดสอบความเร็วสูงสุด (VO2 Max) ที่สมบูรณ์แบบ

## 2. BRU Charity Run (10.5 km)
- **วันที่:** 3 ต.ค. 2026
- **เป้าหมาย:** ทดสอบระบบเอาเวลา
- **ความน่าจะเป็น:** สูงมาก (Very High)
- **บทวิเคราะห์:** มีเวลาเตรียมตัว 8 สัปดาห์หลังงาน 5K เป็นการดึงความเร็วมาขยายสู่ความทนทาน (Endurance) คาดหวังการยืนเพซ 3:40-3:45 /km ได้

## 3. Bangkok Marathon (10 km)
- **วันที่:** 10 ม.ค. 2027
- **เป้าหมาย:** วิ่งเล่นๆ (Tune-up Run)
- **ความน่าจะเป็น:** 100% (Guaranteed)
- **บทวิเคราะห์:** เป็นกลยุทธ์ที่ดีในการซ้อมบรรยากาศงานแข่งก่อนงานหลัก 13 วัน โดยวิ่งในระดับ Zone 2-3 (Tapering) เพื่อไม่ให้ร่างกายช้ำ

## 4. Buriram Marathon 2027 (10 km)
- **วันที่:** 23 ม.ค. 2027
- **เป้าหมาย:** ล่า PB (Personal Best)
- **ความน่าจะเป็น:** สูงทะลุเป้า (Extremely High)
- **บทวิเคราะห์:** เป็นเป้าหมายหลัก (A-Race) มีเวลาสร้าง Base ใหม่เต็มบล็อก 12-14 สัปดาห์ สนามเรียบและอากาศเย็น เหมาะกับการทำ Sub 37 หรือ Sub 36 นาที
